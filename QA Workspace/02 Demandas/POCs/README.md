# POCs

Validação de conceito ou análise exploratória. Pode gerar evidências, bugs, melhorias e conclusões técnicas.

Cada POC é uma nota hub com template [[../../Sistema/Templates/Demanda.md|Demanda.md]] (tipo: POC). Segue a mesma esteira das demais demandas: DEV → HML → Concluída.

## Regras

- Conceito ainda não validado → `status: dev`
- Conceito validado com sucesso → `status: resolvido`
- Conceito descartado → `status: descartado`, mover pra `99 Arquivo/`

Ver [[../../Sistema/Contexto/COMO_EU_TRABALHO#POC|COMO_EU_TRABALHO]] para o fluxo completo.
